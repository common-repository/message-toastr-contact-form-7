Display Toastr success fail and others message For Contact Form 7.

== Description ==

Message by Toastr for Contact Form 7 success and failed messages after submitting form Open Toastr in contact form 7.  Wordpress admin there is all setting to customize Toastr Message. 

Using <strong> Message by Toastr for Contact Form 7 </strong> addon you can replace your validation and success messages into beautiful Toastr Message to attract users.

* **[Message by Toastr Demo](https://aimsdevelop.com/contact-us/)** 


== Toastr Message Features ==

* Easy to Use
* Enable/Disable option for Toastr Message
* Enable/Disable Default Contact Form 7 Message
* Customize Success content
* Enable/Disable Toastr Close Button
* Enable/Disable Toastr Progress Bar
* Enable/Disable Prevent Duplicates Toastr Message
* There is 8 Toastr position support
* Customize background color
* Fully responsive and Lightweight
* Compatible in all major browser

== Installation ==
* Make sure you have Contact Form 7 installed
* Upload the Conversational Forms folder to /wp-content/plugins/
* Activate the plugin through the 'Plugins' menu in WordPress
* Navigate to 'Toastr Message' with Contact forms in wp-admin.


== Frequently Asked Questions ==

= Does this Plugin need Contact Form 7 to work or it can work on any website?  =
